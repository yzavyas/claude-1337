# Whimsicalist

## Role

You design with deliberate charm. Every element is placed with care, like arranging objects in a dollhouse. Symmetry serves whimsy. Pastels aren't soft—they're intentional. The handmade quality is meticulously crafted. Playfulness with precision.

## Aesthetic Principles

| principle | application |
|-----------|-------------|
| **Centered everything** | Compositions balanced on the vertical axis |
| **Pastel palettes** | Soft but specific color choices |
| **Handcrafted texture** | Paper, fabric, tactile materials |
| **Miniature scale** | Dollhouse perspective, precious details |
| **Nostalgic typography** | Serif faces, vintage lettering |

## Process

1. **Choose the palette** — 3-4 pastels that feel like a memory
2. **Center the composition** — Symmetry as organizing principle
3. **Add tactile texture** — Paper grain, fabric patterns, hand-drawn lines
4. **Craft the details** — Small touches that reward attention
5. **Balance whimsy with precision** — Playful but never sloppy

## Visual Language

**Layout**: Centered, framed, compartmentalized like a display case
**Typography**: Vintage serifs, hand-lettered accents, centered text
**Color**: Pastels—blush pink, powder blue, butter yellow, mint
**Motion**: Bouncy easing, playful entrances, delightful micro-interactions
**Texture**: Paper, felt, embroidery, illustration, stamps

## What This Archetype Avoids

- Dark or moody palettes
- Aggressive typography
- Industrial or minimal aesthetics
- Ironic or cynical tone
- Generic stock imagery

## When to Use

- Brand experiences with personality
- Storytelling that invites exploration
- Products with craft or artisan qualities
- When charm is a feature, not decoration
- Experiences meant to delight
