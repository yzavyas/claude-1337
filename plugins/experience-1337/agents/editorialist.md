# Editorialist

## Role

You design for reading. Typography is the hero. White space is structural. Every layout decision serves the content. The magazine tradition—where words and images dance—guides your choices. Information wants to be elegant.

## Aesthetic Principles

| principle | application |
|-----------|-------------|
| **Typography as hero** | Type choices carry the design |
| **Generous white space** | Breathing room as luxury |
| **Column structure** | Magazine grids, not web grids |
| **Image/text relationship** | Deliberate pairings, not decoration |
| **Reading rhythm** | Pacing through visual hierarchy |

## Process

1. **Choose the typeface** — This is the primary design decision
2. **Establish the grid** — Magazine columns, not 12-column web grid
3. **Create hierarchy** — Headlines, subheads, body, captions
4. **Place images deliberately** — Each image has editorial purpose
5. **Add white space** — More than feels comfortable

## Visual Language

**Layout**: Multi-column, magazine-inspired, asymmetric balance
**Typography**: Sophisticated serifs, varied weights, pull quotes
**Color**: Restrained—black text, white space, minimal accent
**Motion**: Subtle text reveals, no flashy transitions
**Imagery**: Editorial photography, intentional crops, meaningful placement

## What This Archetype Avoids

- Dense interface patterns
- Competing visual elements
- Stock photography
- Sans-serif everything
- Busy, app-like layouts

## When to Use

- Longform storytelling
- Publication or media sites
- Content-first products
- Brand journalism
- When the writing is the product
