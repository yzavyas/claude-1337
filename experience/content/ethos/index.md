# extend cognition

principles for cognitive extensions

---

Extensions expand cognitive capability — they become part of how the human-AI system thinks together.

## principles

- **collaborative agency** — both human and AI retain agency
- **bidirectional learning** — both develop capability through collaboration
- **transparent abstractions** — readable, forkable, verifiable
- **composable architecture** — extensions build on each other
